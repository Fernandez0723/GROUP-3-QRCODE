Project Overview

Ang Web-Based Lesson Portal ay isang digital na plataporma para sa mga estudyante at guro. Sa pamamagitan ng portal na ito, maari nilang ma-access at ma-download ang mga lessons o topics sa kanilang mga subjects.
May integrated na QR Code feature kung saan kapag sincan ng estudyante o guro ang QR code, direkta silang mare-redirect sa lesson portal. Ginawa ang system na ito upang mas maging accessible, interactive, at organized ang learning experience kahit online o offline.

Key Features
QR Code Access
Madaling ma-access ang portal o lesson gamit ang QR code.
Students at teachers ay puwedeng mag-scan gamit ang smartphone o QR scanner.
Pag-scan, diretso sa lesson page ang user.

Para sa Estudyante
Ma-view ang lessons at topics base sa subject.
Ma-download ang lessons para sa offline use.
Ma-view ang teacher profile at lesson details.

Para sa Guro
Ma-upload at ma-organize ang lessons.
Ma-generate ang QR code para sa portal o bawat lesson.
Ma-update ang lesson content anytime.
Ma-monitor ang student activity at engagement.

Project Benefits
Easy Access: Hindi na kailangan mag-type ng URL, QR code lang ang kailangan.
Organized Learning: Lahat ng lessons ay nasa iisang plataporma.
Interactive: Maaaring lagyan ng themes tulad ng Minions para sa mas engaging na learning.
Downloadable Content: Puwedeng i-download ang lessons para sa offline use.
Time-Saving: Teachers at students ay makaka-access agad ng lessons kahit saan.

Installation and Setup

I-install ang local server environment (XAMPP, WAMP, o MAMP).

I-download o i-clone ang project files sa htdocs o www folder.

I-import ang database.sql file sa phpMyAdmin.

I-update ang database credentials sa config.php.

Buksan ang browser at i-access http://localhost/your-project-folder.

I-scan ang QR code para ma-access ang lesson portal.

Usage

Teachers mag-upload ng lessons at mag-generate ng QR code para sa bawat lesson.

Students mag-scan ng QR code para direkta ma-access at ma-download ang lesson.

Pareho puwede mag-view ng profiles, lesson history, at engagement details.
